Develop#

Welcome to the Terra developer documentation. If this is your first time building on Terra and you want to set up your environment, use the quickstart guide. Use the how-to guides to learn about using terrad, the command line interface and node daemon for the Terra network, and LocalTerra, a local testing environment. Terra.js and Terra.py are the Javascript and Python SDKs for Terra.
